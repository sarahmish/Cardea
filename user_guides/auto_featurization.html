Featurizer¶

Cardea automatically generates features using the Featuretools package, specifically, the Deep Feature Synthesis (DFS) algorithm to generate a feature matrix from a given dataset. Aiming to fully automate this process, it determines the focus values of the automated feature engineering task using the target entity, and label times of the prediction problem created by Data Labeler.

Once you featurize the data, you will obtain a feature matrix, where each row pertains to a specific instance_id defined in the label_times, and a collection of calculated features.

Featurizing Demo¶

We can continue on our example walkthrough and generate futures on the Missed Appointment dataset.

In [1]: from cardea import Cardea

In [2]: cardea = Cardea()

In [3]: cardea.load_entityset(data='kaggle')

In [4]: label_times = cardea.select_problem('MissedAppointment')

In [5]: feature_matrix = cardea.generate_features(label_times[:1000])
